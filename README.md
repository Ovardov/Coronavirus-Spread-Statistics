## Application Configurations
1. Type in the terminal the following in root directory:
```bash
npm install
npm start
```
2. Make .env file and set your Google Api Key -> REACT_APP_GOOGLE_MAP_API_KEY = 'Your key'

3. Enjoy it!

## Technology stack
- React.js
- SASS
